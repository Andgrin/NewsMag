# NEWS MAGAZINE

Instal json-server if need:
```
npm install -g json-server
```


For Run json-server in:
> /jsonData
```
$ json-server --watch db.json --port 3004
```


For running project just run Gulp, it will create local server on port 8080
```
$ gulp
```
